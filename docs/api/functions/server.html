Server Queries¶

serverCreated¶

serverCreated()¶

Returns True if a Server object is already created, otherwise, returns False.

>>> print(serverCreated())
False
>>> s = Server()
>>> print(serverCreated())
True

serverBooted¶

serverBooted()¶

Returns True if an already created Server is booted, otherwise, returns False.

>>> s = Server()
>>> print(serverBooted())
False
>>> s.boot()
>>> print(serverBooted())
True
